Note to all Designers
  • Make sure to design the banner in 445px or download the PSD file here.
  • Image should be in minimum of Aspect Ratio of 16:9 (1920 x 1080).
    Calculate the aspect ratio here.
  • Image file size should not exceed to 150KB.
  • If the image has overlay, apply an overlay (in your design) with an opacity of 50%. Also please use the colors within our palette. But export the image without the overlay.
Note to all Developers
  • Please maximize our class names provided below.
  • If there are styles that is not within the provided styles, you can over-ride with your own styles on the custom stylesheet that is dedicated only on the page you are working on.
Note: To view the basic class names to be added on subpages please see Banner Page.
