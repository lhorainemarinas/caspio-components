Note to all Designers
  • Make sure to design the banner in 445px or download the PSD file here.
  • Image should be in minimum of Aspect Ratio of 16:9 (1920 x 1080).
    Calculate the aspect ratio here.
  • Image file size should not exceed to 150KB.
  • If the image has overlay, apply an overlay (in your design) with an opacity of 50%. Also please use the colors within our palette. But export the image without the overlay.
Note to all Developers
  • Please maximize our class names provided below.
  • If there are styles that is not within the provided styles, you can over-ride with your own styles on the custom stylesheet that is dedicated only on the page you are working on.
.
Tag Class Names Description
<section> .subpage-banner A mandatory class name for all subpage banners.
  .b445 Sets the height of into 445px
  .center-content Sets the position of the texts inside the banner into Center
  .left-content Sets the position of the texts inside the banner into Left
  .right-content Sets the position of the texts inside the banner into Right
  .overlay-black Adds a black overlay with 50% opacity to make the text readable.
  .overlay-darkest-blue Adds a very dark blue overlay with 50% opacity to make the text readable.
  .overlay-dark-blue Adds a dark blue overlay with 50% opacity to make the text readable.
  .overlay-blue Adds a blue overlay with 50% opacity to make the text readable.
  .overlay-orange Adds a orange overlay with 50% opacity to make the text readable.
  .overlay-green Adds a green overlay with 50% opacity to make the text readable.
  .fix Sets the subpage banner into a parallax effect.
<p> .w960 Sets the width of the sub description into 960px
  .w920 Sets the width of the sub description into 920px
  .w900 Sets the width of the sub description into 900px
